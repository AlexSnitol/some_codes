Аттестация. Учебная практика

ПЕРВЫЙ ЗАПУСК

При запуске программы перед вами появится форма авторизации.
Для того что бы авторизоваться с правами администратора, необходимо ввести в поля:
 - Логин: admin
 - Пароль: admin
Что бы авторизоваться с правами обычного пользователя, необходимо ввести в поля:
 - Логин: manager
 - Пароль: manager

Изменить, добавить или удалить пользователей может только пользователь с правами администратора.
Для того что бы начать работу с пользователями, войдите в систему под администратором и выберите в пункте меню "Таблицы" подпункт "Аккаунты".

Параметры шрифта по умолчанию:
 - Шрифт: Times New Roman
 - Размер: 12 пт.
 - Начертание: Обычный

Остальная информация содержится в файле-справке attestation_practice.chm, открыть который вы сможете через саму программу выбрав в пункте меню "Справка" - "Просмотр справки".
